установить гит.

0)перейти в папку с проектом

1) настройка соединения:
(первый[один] раз)

git init
git config --global user.email "workkbox@gmail.com"
git config --global user.name "workkbox"
git remote add origin https://workkbox/probe.git

2)
-если первый раз будешь скачивать, то сделать команду "git clone https://github.com/workkbox/probe.git"
-чтобы добавить изменения на гит(закачать), написать команды
	-git add .      		(добавить все файлы)
	-git commit -m 'коммент' 	(подготовить к пушу)
	-git push -u origin master	(закачать изменения на гитхаб)


-чтобы получить изменения(допустим ты что-то изменит на ноуте и хочешь обновить на компе)
-на компе ты сделал пуш, последний раз
-переходишь в папку с проектом и пишешь git pull

Комп-Гитхаб-ноут
Ноут-гитхаб-комп

если ты что-то где-то изменил, то всегда в конце делай пуш.
если ты сел за комп, всегда делай пулл(и ноут в том числе)

проверка статуса локальных изменений git status
